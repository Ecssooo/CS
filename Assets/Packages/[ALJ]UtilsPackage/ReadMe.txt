- Raycast 2D : GameObject Player

Create custom raycast : Line, Circle, Box
Can change : Size, direction, offset, layer affect


- TriggerZone2D / TriggerZone3D : GameObject TriggerZone

Trigger Event if an object with a specific component trigger the BoxCollider
Event can be invoked : On One Frame or On Time
On One Frame : Invoked when trigger
On Time : Invoked while collider is triggered. Cooldown customisable.

Component who reacted with trigger can be customised.
